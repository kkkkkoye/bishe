# 农产品微信小程序项目架构说明文档

## 一、项目整体介绍

这是一个**农产品销售微信小程序**项目，你可以把它理解为一个"网上农产品超市"。它的功能包括：
- 用户可以在微信上浏览和购买农产品
- 商家可以管理商品、查看订单
- 管理员可以管理整个系统

---

## 二、项目组成部分（像一栋三层楼）

这个项目由**三个主要部分**组成，就像一栋三层楼：

```
                    ┌─────────────────┐
                    │   微信小程序     │  ← 第三层：用户直接在微信里用
                    │   (mp-weixin)   │
                    └────────┬────────┘
                             │
                    ┌────────┴────────┐
                    │   后端服务       │  ← 第二层：负责处理数据和业务逻辑
                    │ (Spring Boot)   │
                    └────────┬────────┘
                             │
          ┌──────────────────┼──────────────────┐
          │                  │                  │
   ┌──────┴──────┐   ┌───────┴──────┐   ┌──────┴──────┐
   │ 数据库       │   │ 后台管理界面   │   │ 用户H5页面  │
   │ (MySQL)     │   │ (Vue Admin)  │   │ (Vue Front) │
   └─────────────┘   └──────────────┘   └─────────────┘
   
← 第一层：存储数据   ← 管理员用电脑打开  ← 手机浏览器访问
```

---

## 三、各部分详细说明

### 1. 微信小程序 (mp-weixin 文件夹)

**是什么？**
这是用户在微信里直接打开的应用，就像在微信里打开一个网页一样。

**技术栈：**
- 微信小程序原生开发
- WXML（类似HTML）
- WXSS（类似CSS）
- JavaScript

**包含哪些功能页面？**
- `pages/login/` - 登录页面
- `pages/center/` - 个人中心
- `pages/yonghu/` - 用户管理
- `pages/rexiaonongchanpin/` - 热销农产品（商品列表和详情）
- `pages/chanpinfenlei/` - 产品分类
- `pages/shop-cart/` - 购物车
- `pages/shop-address/` - 收货地址
- `pages/news/` - 新闻资讯
- `pages/discussrexiaonongchanpin/` - 商品评论

---

### 2. 后端服务 (springbootw1eo4 文件夹)

**是什么？**
这是整个系统的"大脑"，负责处理所有业务逻辑。它管理数据、响应小程序和网页的请求。

**技术栈：**
- Java 语言
- Spring Boot 框架（简化Java开发）
- MySQL 数据库（存储数据）

**核心功能模块：**

| 模块名称 | 功能说明 |
|---------|---------|
| YonghuController | 用户管理（注册、登录、信息修改） |
| RexiaonongchanpinController | 热销农产品管理（增删改查） |
| ChanpinfenleiController | 产品分类管理 |
| CartController | 购物车管理 |
| OrdersController | 订单管理 |
| AddressController | 收货地址管理 |
| NewsController | 新闻资讯管理 |
| StoreupController | 收藏夹管理 |
| ChatController | 客服聊天功能 |
| FileController | 文件上传下载 |

**后端代码结构（Java）：**
```
src/main/java/com/
├── controller/    ← 控制器：接收请求，调用服务
├── service/       ← 业务逻辑：处理具体业务
├── dao/           ← 数据访问：操作数据库
├── entity/        ← 实体：代表数据库中的表
├── utils/         ← 工具类：辅助功能
└── interceptor/   ← 拦截器：安全验证
```

---

### 3. 后台管理界面 (admin 文件夹)

**是什么？**
这是给**商家/管理员**用的网页，在电脑上打开，类似于"老板的后台"。

**技术栈：**
- Vue 2（前端框架）
- Element UI（界面组件库）
- Axios（网络请求）

**功能页面：**
- 用户管理
- 农产品管理（发布商品、修改价格等）
- 分类管理
- 订单管理
- 购物车管理
- 评论管理

**在哪里找？**
```
springbootw1eo4/src/main/resources/admin/admin/
├── src/
│   ├── views/          ← 页面组件
│   │   └── modules/   ← 各个功能模块
│   ├── components/    ← 可复用组件
│   └── router/        ← 路由配置
├── package.json       ← 项目配置
└── vue.config.js     ← Vue配置
```

---

### 4. 用户 (front 文件夹端H5页面)

**是什么？**
这是手机浏览器也能访问的网页版本，和微信小程序功能类似。

**技术栈：**
- Vue 2
- 一些UI组件库

## 四、数据是如何流动的？

以"用户查看商品列表"为例：

```
用户打开微信小程序
       ↓
小程序发送请求："给我商品列表"
       ↓
后端服务收到请求
       ↓
后端查询数据库
       ↓
数据库返回商品数据
       ↓
后端把数据发给小程序
       ↓
小程序显示商品列表给用户
```

## 五、如何启动这个项目？

### 1. 启动后端服务（Java）
```bash
cd springbootw1eo4
mvn spring-boot:run
```
服务启动后运行在端口 8080

### 2. 启动后台管理界面
```bash
cd springbootw1eo4/src/main/resources/admin/admin
npm install
npm run serve
```
启动后访问 http://localhost:8081

### 3. 启动微信小程序
用微信开发者工具打开 `mp-weixin` 文件夹

## 七、常见问题

**Q: 为什么有小程序又有H5页面？**
A: 小程序只能在微信里用，H5页面可以用任何浏览器打开，扩大用户覆盖范围。

**Q: 管理员怎么使用后台？**
A: 在浏览器输入 http://localhost:8081 ，用账号密码登录后即可管理。

**Q: 数据存在哪里？**
A: 存在MySQL数据库里，这是一个专门存储结构化数据的软件。
